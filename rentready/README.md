# RentReady - Vehicle Rental Intake App

A simple vehicle rental booking system with customer intake form and admin dashboard.

## Features
- Customer booking form with availability calendar
- Automatic pricing (weekday/weekend rates)
- Admin dashboard with sorting, filtering, search
- CSV export
- PIN-protected admin access (default: 1234)

## Vehicles
- Minivan (9 passengers): $95/day, $175/weekend
- Transit Van (12 passengers): $120/day, $220/weekend
